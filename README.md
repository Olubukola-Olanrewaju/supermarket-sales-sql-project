 Supermarket Sales SQL Project

 Project Overview

This project analyzes supermarket sales data using SQL to generate business insights.

 Tools Used

SQL (MySQL)

Dataset

Supermarket sales dataset from Kaggle

Key Insights

* Giza and Alex branches have the highest customer ratings
* Cash and E-wallet are the most used payment methods
* Some product lines generate more revenue than others

 Conclusion

This project helped me improve my SQL skills and understand data analysis.


 Recommendations

Based on the analysis of the supermarket sales data, the following recommendations can be made:

- Focus on top-performing product lines: Products such as Food and Beverages show strong sales and should be prioritized in stock and promotions.  
- Boost underperforming branches: Branches with lower sales can benefit from targeted marketing, promotions, or reviewing inventory levels.  
- Leverage seasonal trends: Certain months show higher sales for specific product lines. Planning marketing campaigns and inventory ahead of these periods can increase revenue.  
- Payment method insights: Encourage popular payment methods to make transactions smoother for customers, and review the less used methods to see if improvements are needed.  
- Bundle strategies: Consider bundling top-selling items with slower-moving products to boost overall sales.  

These recommendations are intended to guide business decisions and maximize sales based on the data analysis performed in this project.
